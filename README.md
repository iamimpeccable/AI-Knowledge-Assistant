# AI Knowledge Assistant

A Retrieval-Augmented Generation (RAG) application built with Python, LangChain, Gemini, and FAISS.

## Features

- Document loading
- LangChain text chunking
- Gemini embeddings
- FAISS vector database
- Semantic search
- Retrieval-Augmented Generation (RAG)
- Prompt engineering for grounded answers

## Tech Stack

- Python
- Google Gemini API
- LangChain
- FAISS
- NumPy
